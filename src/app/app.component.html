Always open

The always open `mat-datepicker` works as a modal which isn't what we want.
This example resorts to using an undocumented `mat-calendar` component instead.
`mat-calendar` is the component datepicker uses, just not in a modal and not tied to an input.
Available props can be found in the source code.
It handles navigation but select events need to be manually handled.
